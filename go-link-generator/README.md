# Go-Link-Generator

A production-ready Go REST API built with **ECHO** and clean architecture.

## Stack

| Layer | Technology |
|---|---|
| Framework | Echo |
| Databases | PostgreSQL |
| ORM |Raw SQL |
| Logger | Zap |
| Auth |None |
| Tracing |OpenTelemetry |
| Metrics |Prometheus |

## Project Structure

## Getting Started

1. Copy and edit environment config:
   ```bash
   cp config/config.local.example.yaml config/config.local.yaml
   # Edit config.local.yaml with your credentials
   ```

2. Start infrastructure:
   ```bash
   docker compose up -d
   ```

3. Run the app:
   ```bash
   make run
   ```

4. Health check:
   ```
   GET http://localhost:8080/health
   ```

5. API docs (Scalar UI):
   ```
   GET http://localhost:8080/docs
   ```
   Swagger JSON:
   ```
   GET http://localhost:8080/swagger/doc.json
   ```
